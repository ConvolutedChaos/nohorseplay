E-Dog OS Changelog

New updates and improvements.

v3.1.3 April 5, 2026
  • New Added the IDE, which can be used to write apps for E-Dog OS
  • New Added hidden files in the file explorer
  • Fix Fixed the title overlapping with the buttons (Aero Theme)
  • Fix Fixed the bottom bar overlapping with the files (File Explorer)
  • Fix Fixed Writer, Paint, and Settings not having any maximize buttons
  • Fix The preview would not expand when maximized (Markdown Viewer)
  • Improved Settings now supports uploaded wallpapers over 3MB
  • Fix Fixed a find-replace error (Writer)
  • Fix Fixed a URL error (Markdown Viewer)
  • Fix Fixed an error where the file explorer would not load sometimes.
  • Improved If you run rm -r /, the system will now ask you.
  • Fix Fixed a problem where more than one line would appear in the context menu
  • New The file explorer now has an icon
  • Fix Wrong password now shows an error message on the login screen
  • Fix Fixed lots of problems with the login screen in the code
  • Fix Fixed a bug in Writer
v3.1.2 April 4, 2026
  • New Added all-purpose rm command in the terminal
  • New Added USB support (Go to Settings)
  • New Added multiple monitor support
  • New Added recycle bin support
  • Improved Improved custom app support
  • Improved Improved zip file opening, will now open as a new volume in the file manager. The old archive manager still exists.
  • New Windows can now be maximized via a maximize button or by double-clicking the title bar. Dragging a maximized window restores it.
  • New Added a changelog app.
