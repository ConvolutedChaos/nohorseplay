HTML Tutorial

Have you ever wanted to learn how to make a website? Here's a tutorial on how to make a website using HTML!

Learn HTML

1. The Basics

First, you'll need to open our Code Editor in a new tab. (hasn't been updated in a while, but it still works.)


Finished? Ok, let's say you wanted to make a page for your shopping list. Type the following into the code editor:

What I need
Eggs
Milk
Bread


Let's see the results:



If you clicked "Run" in the corner, you'll notice that our list is ruined, and everything is collapsed into one line!

HTML can't tell if you want a line break or not, if you just press Enter on your keyboard. To us, it looks like a line break, but it looks no different to HTML.

So, we can use the <br> tag. You can tell what HTML tags are because of the angle brackets (< and >). So, to fix our shopping list, we can use the <br> tag to add line breaks:

What I need <br>
Eggs <br>
Milk <br>
Bread <br>


Let's see the results:



Perfect! When you run the code, you'll see each item on a new line.
The <br> tag is a self-closing tag, meaning it doesn't need a closing tag like most other tags, like <video> </video>.

To make it fancier, we can use the <h1> tag for a bigger title and <ul> with <li> tags for a proper list:

<h1>What I need</h1>
<ul>
  <li>Eggs</li>
  <li>Milk</li>
  <li>Bread</li>
</ul>

Tip: <ul> stands for unorganized list, and <li> stands for list item.


Let's see the results:



If you typed in everything right, you should've seen a nicely formatted list with a bold title.
