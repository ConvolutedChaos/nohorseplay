How To Learn HTML

HTML is the programming language that empowers the web today. Being invented in 1993 by Tim Berners-Lee, HTML has gone through multiple changes.
HTML elements have come and gone, and old ways of styling pages aren't supported anymore.

But here you will learn the basics of HTML so you can create a cool looking page yourself.

Note:

Cross-platform means that a piece of software is available for multiple platforms, mostly Windows, Mac OS, and Linux.

First steps

Code editors

Though there are many options out there, I use Visual Studio Code. Some other cross-platform options are:

  • Visual Studio Code
  • Sublime text (Fee / $99)
  • VSCodium
  • Notepad++ (for Windows)
To make a new HTML document, we go to File → New File, or find something similar to this. Let's create a nice looking page about Rocky Mountain National Park. Paste the following code into your empty file The code is shown below: 
            Rocky Mountain National Park is an American National Park 55 miles away from Denver, Colorado. RMNP is mostly based on the Rocky Mountain Range.
            

            

            Specs:
            

            

            -Act was signed by President Woodrow Wilson on January 26, 1915
            -The Civilian Conservation Corps built the main automobile route, Trail Ridge Road, in the 1930s.
            -Thousands of acres in size
Make sure to save your brand-new document as a HTML file. Once you have that done, double-click on it and it should open in your browser.

Oh no! The browser clearly skimmed across all of our careful formatting and just spat it out in one big block of text!
Here's a screenshot of mine:

We clearly put in a paragraph at the top there, and we want it back!
We can use the <p> tag to mark the beginning and end of paragraphs. First, we make the opening and closing tags like this: 
            <p>This is a paragraph.</p>
HTML will always ignore line breaks in your code. We can get around this by using the <br/> tag. "br" is short for break, as in line break.. Also, if you put in more than one space in next to each other, HTML will omit the other ones so that there only appears to be one.
We can fix this by using &nbsp; in our code. Let's look at this in action: 
            <p>Hello world!</p>
            <p>Hello &nbsp;&nbsp;&nbsp;&nbsp;&nbsp; world!<p>
